FAQ -- Foire Aux Questions

1. Comment passer une commande ? Pour commander un jeu sur notre site, parcourez notre catalogue, ajoutez les jeux souhaités à votre panier, puis suivez les étapes de paiement. Assurez-vous de vérifier les informations de commande avant de finaliser l'achat.
2. Quels sont les modes de paiement acceptés ? Nous acceptons plusieurs modes de paiement, y compris les cartes de crédit (Visa, Mastercard), PayPal, et les paiements par portefeuille électronique. Tous les paiements sont sécurisés.
3. Quand vais-je recevoir mon jeu après l'achat ?
  • Jeux numériques : Vous recevrez un email contenant le lien de téléchargement et le code d'activation immédiatement après l'achat.
  • Jeux physiques : La livraison prend entre 3 à 7 jours ouvrables, en fonction de votre localisation et du mode de livraison choisi.
4. Puis-je annuler une commande ? Oui, les commandes peuvent être annulées si elles n'ont pas encore été expédiées. Pour les jeux numériques, les annulations ne sont possibles que si le code d'activation n'a pas été utilisé. Veuillez nous contacter dès que possible pour demander l'annulation.
5. Comment demander un remboursement ?

Nous offrons des remboursements sous certaines conditions :

  • numériques : Les remboursements sont possibles dans un délai de 14 jours si le code n'a pas été activé.
  • physiques : Retournez le jeu sous 14 jours dans son emballage d'origine non ouvert pour être éligible à un remboursement.

6. Que faire si je rencontre un problème avec l'activation de mon jeu ? Si vous rencontrez des difficultés pour activer votre jeu, assurez-vous d'abord que le code est correctement saisi et vérifiez la compatibilité de votre appareil. Si le problème persiste, contactez notre support client pour obtenir de l'aide.
7. Comment fonctionne la livraison internationale ? Nous proposons la livraison dans plusieurs pays, avec des frais supplémentaires et des délais de livraison variables selon votre localisation. Veuillez consulter notre page de livraison pour connaître les options disponibles pour votre pays.
8. Offrez-vous des réductions pour les achats groupés ou les abonnements ? Oui, nous avons des offres spéciales pour les achats groupés et des remises pour les abonnés. Abonnez-vous à notre newsletter pour recevoir les offres en avant-première.
9. Que se passe-t-il si mon jeu est endommagé à la livraison ? Si votre jeu arrive endommagé, contactez notre service client dans les 48 heures suivant la réception avec des photos de l'état du produit. Nous vous guiderons sur la procédure de retour et de remplacement.
10. Où puis-je consulter les avis sur les jeux ? Chaque jeu a une section "Avis des utilisateurs" sur sa page produit. Vous pouvez lire les avis d'autres acheteurs pour vous aider à faire votre choix.
11. Mon compte utilisateur est bloqué, que faire ? Si votre compte est bloqué, cela peut être dû à plusieurs raisons, comme des tentatives de connexion échouées. Contactez notre support client pour résoudre le problème et récupérer l'accès à votre compte.
12. Comment rester informé des nouvelles sorties et promotions ? Abonnez-vous à notre newsletter ou suivez-nous sur nos réseaux sociaux pour être informé des dernières sorties, des promotions exclusives et des événements spéciaux.
13. Que faire si mon jeu n'est pas compatible avec mon appareil ?

Vérifiez toujours la configuration requise pour chaque jeu avant l'achat. Si votre jeu n'est pas compatible, nous ne pourrons malheureusement pas offrir de remboursement une fois le code d'activation utilisé.
